OidcClientDemo.WPF
=====================================

This tool is able to connect to [OpenID Connect](https://openid.net/specs/openid-connect-core-1_0.html) providers and ask for tokens.

Features
------------------------------
- [x] Authorization Code flow
- [x] Listen for HTTP requests in custom port
